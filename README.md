# ARPChat
Chat over ARP!

*Now with AES128!*

More of a proof-of-concent thing really.
Works on WiFi client isolated networks.

To start it:
<code>./run_chat.sh</code>

The script installs the following packages:
<code>screen<br>
  tcpdump<br>
  python3<br>
  python3-pip<br>
  scapy<br>
  pycryptodome</code>
